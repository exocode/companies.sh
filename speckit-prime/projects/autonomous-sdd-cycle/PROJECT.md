Use this project when a user wants the company to take a feature idea through the full Spec Kit lifecycle.

Default cycle:

1. Establish or update the project constitution.
2. Specify the feature.
3. Clarify underspecified areas.
4. Generate and review the checklist.
5. Plan the implementation with small vertical slices.
6. Generate tasks and refine oversized chunks.
7. Analyze artifact consistency.
8. Implement approved tasks.
9. QA the result and loop back only to the earliest affected phase.
